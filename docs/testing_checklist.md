# Afro Ludo 真机测试清单

## 测试环境要求
- Android 真机（API 28+）
- 安装最新构建的 APK
- 清除应用数据（首次安装测试）

---

## 测试场景 1：完整对局流程
**路径**: 主菜单 → Play Ludo → 设置 2 人局（1 人类 + 1 AI）→ 完整打完一局

### 检查项
- [ ] 主菜单正常显示，所有按钮可点击
- [ ] "Play Ludo" 进入游戏设置界面
- [ ] 可设置玩家数量（2-4人）
- [ ] 可设置人类/AI 玩家类型
- [ ] 游戏棋盘正确渲染（15×15 格子）
- [ ] 16 颗棋子正确显示在初始位置
- [ ] 骰子动画正常播放
- [ ] 棋子可点击选中
- [ ] 可移动棋子高亮显示
- [ ] 棋子移动动画流畅（非瞬移）
- [ ] 吃子逻辑正确（撞子返回基地）
- [ ] 棋子到家逻辑正确
- [ ] 胜负判定正确（4颗棋子到家获胜）
- [ ] 游戏结束界面显示获胜者
- [ ] 可返回主菜单或重新开始

### 预期结果
- 无崩溃、无卡顿
- 所有动画流畅（30fps+）
- 音效正确播放（骰子、移动、吃子、到家、胜利）
